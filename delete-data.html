Request Data Deletion

If you would like your personal data deleted from our systems, please follow the steps below:

  1. Go to our GitHub repository: Open Issues Page
  2. Create a new issue with the title: Request for Data Deletion
  3. In the issue body, please include your user ID or email address used with the app (if applicable).

We will process your request and confirm once the data has been deleted.

Thank you for using our application!
